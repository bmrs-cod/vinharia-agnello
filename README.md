Vinharia Agnello

Este projeto consiste no desenvolvimento de um site para a Vinharia Agnello, com o objetivo de apresentar a empresa, seus produtos, galeria de imagens e formas de contato.

O site foi desenvolvido utilizando HTML e CSS, aplicando conceitos de estrutura semântica, navegação entre páginas e estilização.

-----------------------------------------------------------

Estrutura do Projeto:

* *index.html* → Página inicial
* *historia.html* → História da empresa
* *produtos.html* → Lista de vinhos (com tabela)
* *galeria.html* → Imagens dos produtos
* *contato.html* → Formulário de contato
